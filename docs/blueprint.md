# **App Name**: King Churros

## Core Features:

- Parallax Scroll Background: Implement a full-screen WebP frame sequence background in the hero section that plays forward on scroll down and reverses on scroll up, creating an immersive cinematic effect. The animation should smoothly connect to user scroll.
- Hero Section Content Overlay: Overlay brand logo, a bold title, subtitle, a short description, two call-to-action buttons (ORDER NOW and VIEW MENU), a decorative number '01' with a divider, and monochrome social icons, all positioned to allow clear visibility over the dynamic background.
- Cinematic Pre-loading Screen: Display a brand-centric loading screen with a horizontal progress bar (and optional percentage) that tracks the loading of the WebP frame sequence, preventing flicker and only revealing the hero section upon full preload.
- Sticky Navigation: Provide a persistent top navigation bar with the brand logo and scroll links (Product, Ingredients, Nutrition, Reviews, FAQ, Contact) that smoothly navigate to page sections and highlight the active section.
- Dedicated Content Sections: Design distinct sections for Product/Story, Ingredients, Nutrition (with a label-style card), Customer Reviews/Testimonials, and an Accordion FAQ, each conveying premium product details and brand narrative.
- Final Call-to-Action & Footer: Conclude the page with a compelling 'TASTE THE KING' section featuring duplicate ORDER NOW and VIEW MENU buttons, and a clean, dark footer with brand information, utility links, social icons, and copyright details.

## Style Guidelines:

- Color Scheme: Default to a dark, cinematic theme with charcoal tones for backgrounds and white/light gray for text for strong contrast and a premium feel.
- Primary Color: Rich golden caramel (#F2A60D), inspired by churros and cinnamon sugar, used for CTAs, highlights, and key brand elements. (HSL: 40, 90%, 50%).
- Background Color: A very deep, warm charcoal (#171511), providing a near-black, sophisticated base for content, subtly echoing the primary hue. (HSL: 40, 15%, 8%).
- Accent Color: A rich, deep burnt orange (#AD361F) for subtle secondary accents or differentiation. (HSL: 10, 70%, 40%).
- Headline Font: 'Playfair' (serif) for an elegant, high-end, and editorial feel on titles and key headings. Body Font: 'PT Sans' (sans-serif) for clean readability and a modern complement to the serif headlines.
- Utilize minimalist, monochrome line-art social icons that subtly integrate into the dark theme without distracting from the main visuals.
- Full-screen hero section with visually clear central space, layered text on the left, and an editorial-style decorative element on the right, maintaining a premium, open composition. Subsequent content sections are cleanly defined with ample spacing.
- The hero background features a scroll-driven WebP frame sequence, creating a cinematic, smooth animation directly linked to user interaction. Loading progress bar for content preload and smooth scroll for internal navigation.
